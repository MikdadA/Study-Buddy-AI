Propositions

Proposition: a declarative sentence that is either true or false.

Examples:

  • The moon is made of cheese
  • 1 + 0 = 1

Non-Examples:

  • Sit down!
  • x + 1 = 2

Negation: If p = "The earth is round" then ¬p denotes "It is not the case that the earth is round" or "The earth is not round."

Conjunction: If p = "I am at home" and q = "It is raining" then p ∧ q denotes "I am at home and it is raining."

Disjunction: If p = "I am at home" and q = "It is raining" then p ∨ q denotes "I am at home or it is raining."

Implication (conditional): If p = "I am home" and q = "It is raining" then p → q denotes "If I am home then it is raining."

Biconditional: If p = "I am home" and q = "It is raining" then p ↔ q denotes "I am home if and only if it is raining."

Types of Propositions

Tautology

A proposition which is always true.

Example: p ∨ ¬p

Contradiction

A proposition which is always false.

Example: p ∧ ¬p

Contingency

A proposition which is neither a tautology nor a contradiction.

Mathematical Induction Steps

Prove that p(n) is true for all positive integers 'n'

Note: We aren't proving that p(k) is true, we are proving that if p(k) is true then p(k+1) is also true.

  1. Base step: prove that p(1) is true.
  2. Inductive step: prove that p(k) → p(k+1) is true.
    • Assume that p(k) is true for some arbitrary k.
    • Assuming the above is true for some arbitrary k.
    • Assuming p(k) is true is called the inductive hypothesis.

Once steps 1 and 2 are complete, we've proven that ∀n P(n) is true for the domain ℤ⁺.

We've proven ∀k (p(k) → p(k+1))

Thus: (p(1) ∧ (p(k) → p(k+1)))→∀n p(n) with the domain ℤ⁺

Important note:

  • We do not assume that p(k) is true for all positive integers.
  • We only show that if p(k) is assumed to be true then p(k+1) is also true.

Mathematical Induction Example

Prove that n³ - n is divisible by 3 for all n; domain: ℤ⁺.

  1. State the problem mathematically: let p(n)= n³ - n = 3m where m is in ℤ.
  2. Prove p(1) is true.
    • 1³-1=0
    • Can 0 written as 3 x some integer m?
    • 3x0
    • ie, m=0 in the statement 3m
  3. Prove that if p(k) is true then p(k+1) is true for some arbitrary integer k.
    • Inductive hypothesis: k³-3m
    • Calculate p(k+1)
    • (k+1)³ - (k + 1)
    • k³+3k²+3k+1-(k+1)
    • What are we trying to accomplish?
    • k³+3k²+3k+k+1
    • k³+3k²+3k
    • k³=3m
    • 3m+3k²+3k
    • 3(m+k²+k)
    • 3g

Since p(1) is true and if p(k) is true then p(k+1) is true, thus by mathematical induction p(n) is true for all positive integers, n.
